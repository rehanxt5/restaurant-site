# Ember & Oak Restaurant

Premium restaurant landing page showcasing an upscale dining experience.

## Tech Stack

- **React 19** with Vite
- **Playfair Display** & **DM Sans** fonts

## Getting Started

```bash
npm install
npm run dev
```

## Available Scripts

| Command | Description |
|---------|-------------|
| `npm run dev` | Start dev server with HMR |
| `npm run build` | Production build |
| `npm run lint` | Run ESLint |
| `npm run preview` | Preview production build |
